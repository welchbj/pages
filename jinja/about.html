What is this site?

When learning about a new topic or studying for an exam, I like to compile my notes and findings into Markdown documentation. I find that the best way to ensure mastery of a topic is having the ability to explain it to others.


Because this site is really just for me to have a place to reference my previous notes and code snippets, I have disabled search engine indexing site-wide. If you have still come across this site, feel free to poke around. I have done my best to reference the sources from which I find my information, and they are probably more worth your time than my notes on here.

How is this site built?

This site is an open source project. It uses a combination of Jinja templates and Markdown source files to generate a static HTML site, hosted (for free!) on Netlify. You can take a look at the source on GitHub.
